Detailed Description

A single projectile fired.

The base object that represents a single bullet in a Danmaku game

Member Function Documentation

void Activate ( )

Activates this instance.

Calling this on a already fired projectile does nothing. Calling this on a projectile marked for deactivation will unmark the projectile and keep it from deactivating.

Implements IPooledObject.

void Deactivate ( )

Marks the Projectile for deactivation, and the Projectile will deactivate and return to the ProjectileManager after finishing processing current updates, or when the Projectile is next updated If Projectile needs to be deactivated in a moment when it is not being updated (i.e.

when the game is paused), use DeactivateImmediate instead.

Implements IPooledObject.

void DeactivateImmediate ( )

Immediately deactivates this Projectile and returns it to the pool it came from Calling this generally unadvised.

Use Deactivate whenever possible. This method should only be used when dealing with Projectiles while the game is paused or when ProjectileManager is not enabled

void MatchPrefab ( ProjectilePrefab  prefab)

Makes the instance of Projectile match the given ProjectilePrefab This copies:

  • the sprite, material, sorting layer, and color from the ProjectilePrefab's SpriteRenderer
  • the collider's size and offset from the ProjectilePrefab's CircleCollider2D
  • the tag and layer from the ProjectilePrefab's GameObject
  • any ProjectileControlBehavior on the ProjectilePrefab will be included as additional IProjectileGroupController that will affect the behavior of this bullet
Parameters
prefabthe ProjectilePrefab to match.
