float TargetDeltaTime
staticget

Gets the expected time between each frame for the current Application.

The expected delta time.

float TargetFPS
staticget

Gets the expected frames per second of the current Application.

Normally this is set to Application.targetFrameRate if it is not 0. If the game is paused via setting Time.timeScale to 0, this evaluates to Infinity

The expected frames per second.
