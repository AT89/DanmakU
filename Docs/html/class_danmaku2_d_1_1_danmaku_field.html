Vector2  location,
float  rotation,
CoordinateSystem  coordSys = CoordinateSystem.View

Spawns a projectile in the field.

If absoluteWorldCoord is set to false, location specifies a relative position in the field. 0.0 = left/bottom, 1.0 = right/top. Values greater than 1 or less than 0 spawn outside of of the camera view.

Parameters
prefabPrefab for the spawned projectile, describes the visuals, size, and hitbox characteristics of the prefab.
locationThe location within the field to spawn the projectile.
rotationRotation.
absoluteWorldCoordIf set to true, location is in absolute world coordinates relative to the bottom right corner of the game plane.
